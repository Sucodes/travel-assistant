# Travel Assistant

To run the project, follow the steps listed below

## Prerequisites

1. NodeJs (version 20) and npm

## Set up the frontend

1. In a brand new terminal, change directory to the frontend directory by running `cd frontend`

2. Install the required packages by running `npm i`

3. Start the frontend server by running `npm run dev`

4. Visit http://localhost:5173 to access the frontend and get started.
